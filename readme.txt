== Installation ==

= Plugin Installation Method: =

1. Go to the WordPress dashboard
2. Got to the **Plugins** > **Add New**
3. Type **"Accordion"** in the search box
4. Click on **"Install"** button
5. After install the plugin **Activate** the plugin

= Installation via Zip file: =

1. Download Accordion block **plugin zip** file from WordPress
2. Go to site **Dashboard** > **Plugins** > **Add New** > **Upload Plugins**
3. Select the Accordion zip file and proceed to install
4. Activate Accordion.

== Frequently Asked Questions ==

= How to Use? =

1. **Add the Accordion Block**:
   - Open the WordPress Block Editor (Gutenberg).
   - Click the "+" button to add a new block.
   - Search for "Accordion" and select it from the block library.

2. **Add Content to the Accordion**:
   - Type your header text directly into the accordion title.
   - Add your desired content (text, images, or other blocks) inside the collapsible section.

3. **Customize the Accordion**:
   - **Toggle Settings**: Use the block settings sidebar to customize the accordion:
     - Choose whether the accordion starts open or closed.
     - Enable or disable multiple sections being open at once.
   - **Style Options**:
     - Adjust font size, background color, and padding.
     - Add icons for the expand/collapse toggle.
   
4. **Preview Your Accordion**:
   - Use the "Preview" option in the editor to see how your accordion looks on the front end.

5. **Save and Publish**:
   - Save your changes and publish the post or page to make the accordion visible on your site.


= What features does the plugin offer? =
Accordion offers a range of features designed to enhance your web design experience. These includes customizable blocks, layouts, patterns,  drag-and-drop interface, advanced styling options, flexibility, responsiveness and more.

= Does Accordion work with all WordPress themes? =
Most certainly all WordPress themes are compatible with the Accordion framework as the plugin is built for maximum compatibility with WordPress itself.
